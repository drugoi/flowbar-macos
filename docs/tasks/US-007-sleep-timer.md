# US-007 Sleep Timer

As a listener, I want a sleep timer so playback stops after a set time when I am using the app for focus or sleep.

## Acceptance Criteria
- User can select a timer duration (for example 15, 30, 60 minutes).
- Timer can be canceled or adjusted while running.
- Playback stops when the timer reaches zero.
- Timer status is visible in the Now Playing section.
